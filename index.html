PDF
Tap "PDF" → sélectionnez votre fichier. Texte extrait automatiquement, même sur les PDFs multi-pages.

EPUB (livres numériques)

Lecture audio
5 réglages ▾
🖊️
Surlignage synchronisé
Activez Surlignage ON pour suivre mot par mot. Curseur Sync : vers "Lent" si trop rapide, vers "Rapide" si en retard.
⚡
Vitesse & Tonalité
Vitesse de 0.5× à 3×. Tonalité de 0.5 (grave) à 2.0 (aigu). Mémorisées par document.
🎙️
Choisir une voix
Les voix ⭐ Google sont les meilleures. Pour les installer : Paramètres → Accessibilité → Synthèse vocale → Google TTS → Installer données vocales.
💡 Sur Microsoft Edge, des voix neurales HD sont disponibles sans installation — recommandé sur Xiaomi.
📖
Mode de lecture
Par phrase — découpe en phrases courtes, plus fiable sur Android.
Par § — paragraphes entiers, plus fluide sur desktop.
📞
Reprise après appel
Si un appel ou une notification coupe la lecture, l'app détecte le retour en premier plan et reprend automatiquement.

Activer le Mode Zen
Tap Mode Zen dans la barre de lecture. Fond noir total, texte centré style téléprompter. Quittez avec ✕ ou swipe vers le bas.

🔧
Dépannage
5 problèmes ▾
🔇
L'audio ne démarre pas
Sur Android, l'audio requiert une interaction. Tap d'abord sur la page, puis sur Lecture. Vérifiez que le volume n'est pas coupé.
💡 L'app relance automatiquement la lecture si elle s'arrête silencieusement (watchdog actif).
⚠️
La lecture s'arrête sans prévenir
Un système de surveillance (watchdog) relance automatiquement la lecture dans les 3.5 secondes si la synthèse vocale s'arrête de façon inattendue.
✅ Corrigé automatiquement
🖊️
Le surlignage est décalé
Ajustez le curseur Sync dans Réglages : vers "Lent" si le texte avance trop vite, vers "Rapide" s'il est en retard.
🎙️
Je ne vois que des voix locales
Sur Xiaomi HyperOS, Chrome ne expose pas les voix Google HD. Utilisez Microsoft Edge pour accéder aux voix neurales HD.
⚠️ Problème connu sur Xiaomi HyperOS + Chrome
🌐
L'import URL échoue
Certains sites bloquent les proxies CORS. Solution : ouvrez la page dans Chrome, sélectionnez tout, copiez et collez dans Saisie manuelle.
