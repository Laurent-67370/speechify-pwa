Aide

Guide d'utilisation de SpeechifyPro.

🚀 Démarrage rapide
1
Importer un document
Tap sur Importer dans la nav → choisissez PDF, TXT, image ou collez une URL d'article web.
2
Lancer la lecture
Tap sur Lire dans la nav → bouton Lecture. L'audio démarre et le texte défile avec surlignage.
3
Reprendre où vous étiez
Votre position est sauvegardée automatiquement. Rouvrez l'app et tap Reprendre sur l'Accueil.
📥 Importer des documents
📄
PDF
Tap "PDF" → sélectionnez votre fichier. Le texte est extrait automatiquement, même sur les PDFs multi-pages.
🌐
URL d'article web
Collez n'importe quelle URL d'article ou blog → tap ⬇️. Le texte principal est extrait et ajouté à la bibliothèque.
💡 Si l'import échoue, le site est probablement protégé par JavaScript. Copiez-collez le texte manuellement.
📷
Image OCR
Photographiez un document papier ou chargez une image. L'OCR extrait le texte automatiquement (connexion requise au premier usage).
Français + Anglais
🔊 Lecture audio
🖊️
Surlignage synchronisé
Activez Surlignage ON pour suivre mot par mot. Ajustez le curseur Sync si le surlignage est en avance ou en retard.
⚡
Vitesse & Tonalité
Vitesse de 0.5x à 3x. Tonalité de 0.5 (grave) à 2.0 (aigu). La vitesse est mémorisée par document.
🎙️
Choisir une voix
Les voix ⭐ Google sont les meilleures. Si vous n'en voyez pas, installez-les via Paramètres → Accessibilité → Synthèse vocale → Google TTS → Installer données vocales.
💡 Sur Microsoft Edge, des voix neurales HD sont disponibles sans installation.
📖
Mode de lecture
Par phrase : découpe le texte en phrases courtes — plus fluide sur Android. Par § : paragraphes entiers — meilleure fluidité sur desktop.
✨ Fonctionnalités avancées
🌑
Mode Zen
Fond noir total, texte centré style téléprompter. Tap Mode Zen → lecture immersive. Quittez avec ✕ ou la touche Échap.
🔖
Marque-pages
Tap Marquer pendant la lecture pour sauvegarder votre position. Retrouvez vos marques dans le panneau qui s'ouvre.
🎯
Objectif quotidien
Sur l'Accueil, tap sur objectif : X min ✏️ pour personnaliser votre cible. La progression est sauvegardée chaque jour.
🔔
Notifications
Une notification s'affiche à la fin de chaque lecture. Autorisez les notifications quand l'app vous le demande.
🔧 Dépannage
❓
L'audio ne démarre pas
Sur Android, l'audio nécessite une interaction préalable. Tap d'abord sur la page, puis sur Lecture. Vérifiez que le volume n'est pas coupé.
❓
Le surlignage est désynchronisé
Ajustez le curseur Sync surlignage dans Réglages. Vers "Lent" si le texte avance trop vite, vers "Rapide" si il est en retard.
❓
Je ne vois que des voix locales
Sur Xiaomi HyperOS, les voix Google HD ne sont pas exposées à Chrome. Essayez Microsoft Edge pour accéder aux voix neurales HD.
❓
L'import URL échoue
Certains sites bloquent les proxies CORS. Solution : ouvrez la page dans Chrome, sélectionnez tout le texte, copiez et collez dans Saisie manuelle.
